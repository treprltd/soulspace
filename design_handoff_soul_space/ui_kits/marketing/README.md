# Soul Space — Marketing UI Kit

The marketing landing page. One long centred column with these sections in order:

1. **MarketingNav** — logo left, Settings + Begin CTA right.
2. **Hero** — eyebrow, serif display, mist tagline, affirmation, gold CTA, fine-print.
3. **ResonanceShowcase** — 2×2 grid of the four resonance phrases (the four branches).
4. **MirrorExample** — three Mirror cards demonstrating the output.
5. **ScopeIsIsNot** — two columns, teal ✓ / muted-red ✕.
6. **ClosingCTA** — "Does the first session earn the second?"
7. **MarketingFooter** — wordmark, fine print, two muted links.

Open `index.html` to view. All copy is verbatim from `src/app/page.tsx`.
